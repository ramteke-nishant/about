System Administrator

Skills

Windows Server Administration, Linux Server Administration, Network Configuration, Security Management, Scripting (Bash, PowerShell), Virtualization (VMware, Hyper-V), Monitoring and Troubleshooting, Active Directory, DNS, DHCP, Group Policy Management, Patch Management (WSUS), Shell Scripting (Bash), Web Server Administration (Apache, Nginx), Database Administration (MySQL, PostgreSQL), System Documentation, Technical Support, Disaster Recovery Planning.

Job Roles and Responsibilities

Windows System Administrator
  • Manage and maintain Windows Server environments including Active Directory, DNS, and DHCP.
  • Implement group policies and manage user permissions.
  • Monitor system performance and ensure high availability.
  • Deploy patches and updates using WSUS and PowerShell scripting.
Linux System Administrator
  • Configure and administer Linux servers (Ubuntu, CentOS).
  • Automate tasks with shell scripting (Bash).
  • Manage web servers (Apache, Nginx) and database servers (MySQL, PostgreSQL).
  • Implement security measures and perform regular updates.
General Responsibilities
  • Provide technical support and troubleshoot issues across Windows and Linux platforms.
  • Collaborate with development teams to deploy applications and services.
  • Create and maintain comprehensive documentation.
  • Participate in disaster recovery planning and execution.
